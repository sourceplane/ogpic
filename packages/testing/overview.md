# Testing

Shared test fixtures and utilities for the Ogpic monorepo. It centralizes the
helpers and fixtures that other packages and apps use to write consistent tests.

## Responsibilities
- Provide reusable test fixtures for the monorepo.
- Provide shared testing utilities and helpers.
- Reduce duplication of test setup across components.

## Delivery
Built and type-checked in the Turborepo; consumed by other components.
