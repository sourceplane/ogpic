# Stack Tectonic

Repo-local Orun composition stack of golden paths for the Ogpic platform. It
bundles the reusable compositions that define how components are delivered via
Orun, and is published to the registry for consumption.

## Responsibilities
- Define the golden-path Orun compositions used across the repo.
- Package those compositions as a versioned publishable stack.
- Publish to `ghcr.io/sourceplane/stack-tectonic`.

## Delivery
Published as an Orun stack to `ghcr.io/sourceplane/stack-tectonic`; dry-run on
dev, release on staging and production.
