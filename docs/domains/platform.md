# Platform

The horizontal foundation the product domains stand on: the API edge, the
event log, configuration/flags/secrets, admin tooling, provider integrations,
the shared libraries, the delivery stack, the infrastructure, and the
verification suites. If it serves every domain and belongs to none, it lives
here.

Systems in this domain: `edge`, `operations`, `platform-libraries`,
`infrastructure`, `quality`, and the `messaging` event log.
